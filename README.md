# SAD-operating-unit-for-VVC-Motion-Estimation
# Author: Vítor Silva da Costa
# Federal University of Pelotas

This architecture was part of my final undergraduate project. This repository contains the architectural implementation of the SAD operating unit for the VVC Motion Estimation. The architecture is described in VHDL and is oriented for low-power SAD calculation. Further details about the project can be found in the .PDF containing the undergraduate thesis, as well in the following papers:

https://doi.org/10.29292/jics.v20i1.964
https://doi.org/10.1109/SBCCI66862.2025.11218658


This architecture was synthesized for TSMC 40nm standard cell technology, utilizing Cadence Genus Synthesis Solution.
